# Walkabout Investments LLC Website

This is the static website for Walkabout Investments LLC, a family-owned property rental and investment business in Washington state.

## Project Structure

```
walkabout-investments-static/
├── index.html          # Main website file
├── css/
│   └── style.css      # Custom styles
├── js/
│   └── script.js      # Custom JavaScript
├── img/
│   └── services/      # Service images
└── .github/
    └── workflows/     # GitHub Actions for deployment
```

## Features

- Responsive design using Bootstrap 5
- Modern CSS with animations and transitions
- Smooth scrolling navigation
- Contact information and services
- Optimized for Azure Static Web Apps

## Local Development

1. Clone this repository
2. Open `index.html` in a web browser
3. Or use a local server like Live Server in VS Code

## Deployment

This site is configured for deployment to Azure Static Web Apps with automatic GitHub Actions integration.

## Custom Domain Setup

The site supports custom domain configuration through Azure Static Web Apps:
- Domain: www.walkaboutinvestments.com
- SSL/TLS certificates are automatically managed by Azure
